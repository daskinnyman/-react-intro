# 如何使用這個repo

### 1.複製repo
```
git clone https://github.com/daskinnyman/-react-intro.git
```

### 2. cd -react-intro

### 3. 下載dependencies
```
npm i 
```
### 4. 開始專案
```
npm start 
```
